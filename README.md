# jp-assembly-citizen

市民が日常の「困りごと」「気づき」「提案」を、音声・テキスト・写真・位置情報で気軽に投稿できる OSS クライアントです。投稿は地図上に可視化され、地域の課題を共有しやすくなります。

> このリポジトリは現在 PoC（実証実験版）であり、完成形ではありません。  
> 機能・仕様は今後のフィードバックに基づき改善されます。

jp-assembly プロジェクトのうち、市民の声を収集し、構造化する入力レイヤーを担います。

---

## 目的

- 市民が迷わず投稿できる心理的安全性の高い仕組みをつくる
- 感情的・断片的な市民の声を AI によって構造化し、標準化されたデータとして蓄積する
- 将来的に自治体や地域コミュニティが活用できるデータ基盤をつくる
- プロジェクト全体の方針・標準化は jp-assembly-platform に集約する

---

## PoC（実証実験版）について

この PoC は、市民が地図上で困りごとを投稿し、AI が内容を整理して行政へ届けるまでの最小フローを確認するためのデモ版です。

PoC 動作環境:  
https://jp-assembly-citizen.hito7772000.workers.dev/

### PoC で確認できること

- 地図を起点とした投稿フロー
- AI による市民の声の要約・構造化
- 投稿地点と周辺投稿の可視化
- 行政側の位置情報（庁舎）との重ね合わせ

---

## PoC の動作概要

### 1. 都道府県の選択
利用者は最初に都道府県を選びます。

### 2. 市区町村の選択
選択した都道府県に属する市区町村の一覧が表示され、投稿先の自治体を選びます。

### 3. 地図画面
選択した市区町村の地図が表示され、以下の情報が確認できます。

- 投稿したい場所をタップすると投稿画面へ遷移
- 周辺の投稿（紫のマーカー）を表示
- 自治体庁舎の位置（灰色のマーカー）を表示

### 4. 投稿画面
地図で選んだ地点に対して、困りごとを投稿できます。

- 画面下部の入力欄に困りごとを記入
- 「送信」を押すと AI が内容を解析し、要約と分類を提示
- 投稿内容を確認し、「はい」を選択すると投稿が完了
- 投稿完了後は自動的に地図画面へ戻ります

## 現在の機能（フェーズ1：PoC 範囲）

### 投稿機能
- [ ] 音声入力（AI による文字起こし・要約）
- [x] テキスト入力
- [ ] 写真（最大3枚）
- 位置情報
  - [x] 地図タップ
  - [ ] GPS

### AI による構造化処理
AI が以下を自動で実施します。

- 問題点の抽出
- 解決策（市民が望む方向性）の抽出
- 標準化カテゴリへの分類
- 要約の生成
- DB への構造化保存

### データ保持
- 写真は 50 日間保持後に自動削除
- テキスト・要約・カテゴリ・位置情報・抽出情報は DB に保持

### 表示
- 投稿は地図上に表示
- 投稿者名などの個人情報は扱わない
